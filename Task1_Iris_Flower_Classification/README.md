# Iris Flower Classification

## Objective
The objective of this project is to build a Machine Learning model that can classify iris flowers into three species based on their measurements.

## Dataset
The dataset contains the following features:
- SepalLengthCm
- SepalWidthCm
- PetalLengthCm
- PetalWidthCm
- Species

The target column is **Species**.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Steps Performed
1. Imported required libraries
2. Loaded the dataset
3. Checked dataset information and null values
4. Removed the unnecessary Id column
5. Performed exploratory data analysis
6. Created pairplot and heatmap visualizations
7. Split the dataset into training and testing sets
8. Trained a Random Forest Classifier
9. Predicted species on test data
10. Evaluated model accuracy
11. Generated confusion matrix and classification report
12. Tested the model with a custom sample input

## Model Used
Random Forest Classifier

## Result
The model achieved high accuracy in classifying iris flower species correctly.

## Conclusion
This project shows that Machine Learning can effectively classify iris flowers based on sepal and petal measurements.
